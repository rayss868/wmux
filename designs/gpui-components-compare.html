gpui 시그니처는 딱 2가지 — 이걸 요소마다 변주할 뿐

① 튀어나온 것(버튼·카드·탭·메뉴 항목) = surface fill + inset 0 1px 0 상단 하이라이트. ② 움푹한 것(입력·검색바) = 어두운 배경 + inset 0 1px 2px 그림자 + 포커스 시 0 0 0 3px accent 링. 전부 color-mix(토큰) 기반이라 테마 8종 자동 대응이고, DESIGN.md의 amber 다이어트·hairline·터미널 히어로는 그대로예요. 

/* ① 튀어나온 것 */            /* ② 움푹한 것 */
box-shadow: inset 0 1px 0        background: var(--bg-base);
  rgba(255,255,255,.045);        box-shadow: inset 0 1px 2px rgba(0,0,0,.28);
/* :focus → */ box-shadow: …, 0 0 0 3px color-mix(in srgb, var(--accent) 20%, transparent);

실제 적용은 산재한 ad-hoc Tailwind를 <Button>·<Input>·<Menu> 등 공유 프리미티브로 수렴시키는 리팩터. amber 다이어트를 오히려 개선합니다(카드 hover·검색 토글의 amber 채움 제거).
